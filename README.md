
# Production Forecasting with LSTM

This project forecasts oil & gas production using deep learning (LSTM) and baseline models (Linear Regression, Holt-Winters).

## Notebooks
- `01_data_cleaning.ipynb` – Data cleaning and preprocessing
- `02_exploratory_analysis.ipynb` – EDA: trend, seasonality, outliers

## Data
- Cleaned monthly oil & gas production data (2015–2025)
